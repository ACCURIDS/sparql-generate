We extract a piece of information from the JSON document using the fun:JSONPath binding function.

In this setup the SPARQL-Generate engine will use the air quality JSON report that is explicitly defined in the documentset below. You can edit or delete this document, and add other documents if required.

SPARQL-Generate Binding functions are defined to generate RDF from JSON, XML, HTML, CSV, and plain text. They all take a set of RDF terms as input, and output zero or one RDF term. They all have namespace http://w3id.org/sparql-generate/fn/ with preferred prefix fun.

Remember: binding functions are documented at their URI. The set of available binding functions is available at http://w3id.org/sparql-generate/fn/
