Simple example with iterator function iter:MQTTSubscribe, which connects to a MQTT server, subscribes to some topics, and issues bindings for the topic (first variable) and the message (second variable) when they are received.

  • Param 1: (a String or URL) the MQTT server. Two types of connection are supported tcp:// for a TCP connection and ssl:// for a TCP connection secured by SSL/TLS;
  • Param 2 .. n (optional, Strings): topics to subscribe to.

For SSL configuration (programmatic or using JVM arguments), see the constructor of MqttClient.

Additional connection OPTIONS configuration can be programmatically set. See the API documentation.

The MQTT message (byte[]) is assumed to be a UTF-8 string. This behaviour may also be changed programmatically.
