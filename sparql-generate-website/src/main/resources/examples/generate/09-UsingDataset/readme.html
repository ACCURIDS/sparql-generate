SPARQL-Generate queries are executed against a combination of a RDF Dataset and a DocumentSet

The query below parametrizes the generation of RDF using a graph that contains a set of instances of the following pattern. (Click the edit button to check out what the default graph of the Dataset contains).

?x  map:jsonKey ?key ;
  map:inUcumUnit ?unit ;
  map:observationClass ?obsClass ;
  map:propertyClass ?pClass .
