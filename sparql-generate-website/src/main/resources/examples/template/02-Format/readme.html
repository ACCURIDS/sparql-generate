Using FORMAT statements is the original way defined in STTL to specify a string pattern with text variables (%s) and a list of expressions. Text variables are replaced by the values of expressions.

The syntax of the GROUP statement is as follows:

FORMAT { <PrimaryExpression> ( <TExpression> )+ }
