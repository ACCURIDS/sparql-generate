An example showcasing how does SPARQL-Generate deals with geospatial linked data, like GeoJSON geometry data.
In this example, iter:GeoJSONFeatures iterates and extracts the GeoJSON Features from a GeoJSON object with the type < href="https://tools.ietf.org/html/rfc7946#page-12">FeatureCollection.
fun:GeoJSONGeometry extracts the Geometry member from a Feature object as a wktLiteral
