An example showcasing how SPARQL-Generate deals with geospatial linked data, like GeoJSON geometry data. In this example, you can iterate over a GeoJSON document in two ways, with iter:GeoJSONFeatures or iter:GeoJSON.

  1. iter:GeoJSONFeatures iterates and extracts the GeoJSON Features from a GeoJSON object with the type FeatureCollection.
    • fun:GeoJSONGeometry extracts the Geometry member from a Feature object as a wktLiteral.
  2. iter:GeoJSON returns two iterators:
    • the Geometry member (as a wktLiteral) of each Feature object in the FeatureCollection GeoJSON object.
    • the "properties" member as a JSON object.
