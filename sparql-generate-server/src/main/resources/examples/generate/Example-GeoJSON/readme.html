The query below is a simple example with GeoJSON geometry data.

Iterator function iter:GeoJSON iterates over the features collection of a GeoJSON document, and outputs (1) the Geometry as a wktLiteral, and (2) the Features as a JSON Literal

  • The input is the URI of the GeoJSON document (a URI), or the GeoJSON object itself (a String)
  • As output, the following variables may be bound:
    • Output 1: (wktLiteral) geometry of the feature;
    • Output 2: (json literal) properties of the feature

Binding function fun:GeoJSONGeometry extracts the Geometry member from a Feature as a wktLiteral.

  • The input is the GeoJSON Feature object (a JSON literal)
  • The output is a wktLiteral
