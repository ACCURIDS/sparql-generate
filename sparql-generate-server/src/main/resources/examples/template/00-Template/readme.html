SPARQL-Generate extends a sub-set of the SPARQL-Template language (a.k.a. STTL), and can generate text from RDF and documents.

The TEMPLATE clause is used to specify a text pattern that is instantiated for each solution of the where clause, and optionally joined with a <separator> string. Its syntax is as follows:

TEMPLATE { (<TExpression>)* ( ; separator = <separator> )? }

The <TExpression>s in the text pattern can be of different types, we will overview each of them in this tutorial.

The simplest possible <TExpression> are constants (literals) and evaluable expressions (variables, function calls, etc.).
