Like for SPARQL-Generate, SPARQL-Template queries may be modularized. A SPARQL-Template query can call another SPARQL-Template queries, provided this other query is named and declares parameters.

The binding function http://ns.inria.fr/sparql-template/call-template, abbreviated st:call-template can be used to call a SPARQL-Template queries by its name, and pass it arguments:

  • Param 1: (template): the URI of the SPARQL-Template query;
  • Param 2 .. N : (arguments ... ) the arguments with which the query will be called.
